## 📂 Installation

Clone the repository

```bash
git clone https://github.com/Vedkumar07/<repository-name>.git
```

Navigate into the project

```bash
cd <repository-name>
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
